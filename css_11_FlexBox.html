Basics of CSS Flexbox

Introduction:

Flexbox also called flexible box layout also makes it easier to layout,align and style items in th container while maintaining the responsiveness of the design.
To create a flexbox you need to set the display of the container as flex.
display:flex;

There is a flex container that stores sub elements called flex items

Flex container properties:

  1. Flex direction

    2. It defines in which direction the flex elements would be displayed.It takes values like row column or reverse
    Flex-direction: row; (Default value of flex-direction is row)
