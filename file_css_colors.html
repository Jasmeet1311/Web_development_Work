colors

The first method of defining the colour in the CSS is directly writing the particular colour name.

Its example is #firstPara{ color:blueviolet; /* Color by name */ }

he second way of defining colour is with the help of ‘RGB,’.
The RGB colour range varies from 0 to 255.

The third way is by giving hex colours. However, it is very rarely used.
#thirdPara{ color: white; background-color: #ff4532; /* Color by hex value */ }
